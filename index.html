LAHNA'S CHOCOLATES - PERSONAL PROJECT

React | Redux | Node | Nodemailer | Sass | Auth0 | PostgreSQL

An e-commerce site to make buying hand dipped chocolates easier and more visually pleasing. Uses Sass over CSS for better styling, React routing for better naviagtion from page to page, Redux to store information easier and better throughout the whole website, and PostgreSQL for storing user info, order info, and product info in a database. Uses Nodemailer for contacting the company, and for sending order info to the customer and company.

lahnaschocolates.theaustinwright.com GitHub

PLANAPPUS - GROUP PROJECT

React | Redux | Node | Nodemailer | Passport | Sass | PostgreSQL

A website designed to help developers and designers plan their projects in a more organized way all in one place. Uses React routing to make naviagation fast and easy, Sass in place of CSS for better styling, and Passport for simple and secure user authentication.
